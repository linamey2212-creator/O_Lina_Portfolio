## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/portfolio.git
   ```

2. Navigate into the project directory:

   ```bash
   cd portfolio
   ```

3. Install dependencies (if applicable):

   ```bash
   npm install
   ```

4. Open with live server or Open your browser and go to `http://127.0.0.1:5500/index.html` to view the website locally.
